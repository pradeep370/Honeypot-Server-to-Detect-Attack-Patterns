Setup Guide & Requirements

1) REQUIREMENTS NEEDED

  • Python 3.x
  • pip (Python package manager)
  • Virtual environment (recommended)
  • Cowrie SSH Honeypot
  • Python libraries: geoip2, folium
  • (smtplib and email are built-in, no need to install)

2) GITHUB LINKS

  • Cowrie Honeypot
  • My Project Repository

3) COMMANDS TO FOLLOW


# --- Clone Your Project Repo ---
git clone https://github.com/pradeep370/Honeypot-Server-to-Detect-Attack-Patterns.git
cd Honeypot-Server-to-Detect-Attack-Patterns

# --- Install Cowrie ---
git clone https://github.com/cowrie/cowrie
cd cowrie

# Create virtual environment
python3 -m venv cowrie-env
source cowrie-env/bin/activate

# Install Cowrie dependencies
pip install --upgrade pip
pip install -r requirements.txt

# --- Install Project Dependencies ---
pip install geoip2 folium

# --- Run Cowrie Honeypot ---
bin/cowrie start

# --- Run Your Alert Script (automated email alerts) ---
python3 alert.py

4) TESTING & MONITORING


# --- Connect via SSH (from another machine/VM) ---
ssh root@<your-server-ip> -p 2222
# Example:
ssh root@192.168.1.100 -p 2222

# --- View Live Logs (real-time monitoring) ---
tail -f cowrie/var/log/cowrie/cowrie.json
tail -f cowrie/var/log/cowrie/cowrie.log
